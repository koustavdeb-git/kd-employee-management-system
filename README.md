# Employee Admin Dashboard

React + Vite admin dashboard scaffold for employee management.

## Setup

```bash
npm install
npm run dev
```

## Features

- React + Vite
- Tailwind CSS
- React Router
- Axios base instance
- TanStack Table
- React Hook Form

## Structure

- `src/components` - reusable UI, layout, tables, forms, common components
- `src/pages` - page-level views for dashboard, employees, settings
- `src/services` - API client and service methods
- `src/routes` - router definitions
- `src/layouts` - layout container components
- `src/utils` - formatting and helper utilities
- `src/hooks` - reusable React hooks
- `src/context` - shared application context
- `src/data` - static config and column definitions
